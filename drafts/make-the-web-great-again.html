Make the Web great again

Web 2.0

SNS の走りや集合知を活用、UGC(User generated content)で Web はより素晴らしいものになるのかと思われた

SNS Everywhere

猫も杓子も SNS の時代へ。多様性は個別のクラスタが島宇宙のように存在する闇の間のみに存在するようになった。クラスタ内では承認の経済により多様性は失われ、単一の価値観へと帰結する。

一見クラスタ内では多様性が担保されてるように見えるが SNS のフォロー、フォロワーのグラフから生まれるのは多様性ではなくエコー・チェンバー現象だということがここ数年で明らかになったことだ。

特定の価値観が検証を経ずに部屋の中を響くように共鳴しより強化される。そこに不確定性が挟まる余地というのが非常に少なくなる。そうしたエコー・チェンバー現象の帰結は見ての通りアメリカ選挙戦でのトランプの勝利に繋がり、Facebook 内での女性エンジニアへの差別に繋がった。

そして再度 Web へ

改めて Web とはなんだったのか。

クライアントとサーバをつなぐ WWW として生まれ、HTTP というプロトコルによって成り立つ分散システム。

知識は集約され、人類の進歩になるかと思われたが Facebook のプライバシーへの侵害から再考されるようにオリジナルなきコピーが個体が失われ一つの単一の人格のように扱われるようになってしまった。ここには社会が失われ人間がいない。ディープラーニング、機械学習で扱われる対象も単なるデータであり、そこから生み出される結論は多様性を失っている。ドーキンスから引用し多様性を失った人類はその寿命を早めているのではないかと思う。多様性を取り戻そう。

そういう意味でデイリーポータルやバーグハンバーグバーグなど一見して何をやっているか分からないということは重要なのである。僕らが Web に感じた未来というのはそういった多様性の未来だったのでは？

よく分からないものを作ろう。よく分からない意見を言おう。自分の頭で考えよう。意識的な情報の切断。交換可能な他者にならないこと。
